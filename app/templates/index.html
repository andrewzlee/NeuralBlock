NeuralBlock: An In-Video YouTube Sponsor Detector

Welcome!

This is the project page for NeuralBlock (NB), a neural network built using Keras/Tensorflow that detects in-video YouTube sponsorships. You can find the project on Github. Additionally, this web application provides an API to SponsorBlock, whom supplied the crowd-sourced sponsor labels needed to train NB.

Demo

Submit a video ID and see where the sponsorships are. For true predictions, submit videos published after March 3, 2020. Videos without English captions currently cannot be proccessed.

Sample: https://www.youtube.com/watch?v=QqEuO5im1nA
